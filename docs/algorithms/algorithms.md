# Algorithms Documentation

## Bubble Sort
**Description:** Sorts exercises by calories burned in ascending order.  
**Steps:**  
1. Compare adjacent elements.  
2. Swap if they are in the wrong order.  
3. Repeat for all elements.  
**Time Complexity:** O(n^2)  
**Usage:** Used to sort the workout log for better viewing.

## Linear Search
**Description:** Searches for an exercise by name in the list.  
**Steps:**  
1. Start from the first element.  
2. Check if it matches the target.  
3. Move to the next element.  
4. Repeat until found or end.  
**Time Complexity:** O(n)  
**Usage:** Used to find specific exercises in the log.

## Recursive Average Calculation
**Description:** Recursively calculates the average calories from a list.  
**Steps:**  
1. Base case: if last element, return its value.  
2. Recursive case: add current value to average of rest, weighted.  
**Time Complexity:** O(n)  
**Usage:** Demonstrates recursion for statistical calculations.
